  ## About The Project
  
  
  A Node.js application which allows a user to generate a README.md file after answering a series of prompts.
  
  Built as an assignment for the University of Washington Full Stack Coding Bootcamp. 
  
  
  ### Built With
  
  * JavaScript
  * Node.js
